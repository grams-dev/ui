# Welcome to Grams UI

Grams UI is a collection of reusable UI components developed by the Grams Foundation. The storybook serves as a visual documentation and playground for exploring and testing the various UI elements available in the library. Your project must be compatible with react-i18next, as many of the components rely on internationalization. For convenience, you can use the `@grams-dev/i18n` to work seamlessly with Grams UI.

<div className="subheading">Get Started</div>

```bash
npm install --save @grams-dev/i18n @grams-dev/ui
```

Then, add the i18n instance and the css style to your `index.(js|ts)`

```javascript
import "@grams-dev/i18n";
import "@grams-dev/ui/dist/grams.css";
```
